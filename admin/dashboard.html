TENTANG WEBSITE


Dikembangkan dengan Framework Laravel 7.0


Laravel adalah pengembangan website berbasis MVP yang ditulis dalam PHP yang dirancang untuk meningkatkan kualitas perangkat lunak dengan mengurangi biaya pengembangan awal dan biaya pemeliharaan, dan untuk meningkatkan pengalaman bekerja dengan aplikasi dengan menyediakan sintaks yang ekspresif, jelas dan menghemat waktu.

Fitur Website:

  • Menjalankan navigasi Frontend (User) dan Backend (Admin)
  • Mengeksekusi perintah CRUD saat mengelola data tabel
  • Menghasilkan laporan tabel transaksi dalam bentuk PDF dan Excel

Tampilan website yang ramah pengguna serta modern


Dengan dukungan framework CSS dan HTML Bootstrap, website ini dapat memperoleh tampilan frontend dan backend yang simple dan interaktif.

Versi : 4.1
Tema: Carousel Theme

Dirancang untuk memenuhi kebutuhan proses bisnis yang berjalan di usaha klien


Diadakannya website ini ditujukan untuk mempermudah pelanggan baik pelanggan baru maupun pelanggan sehari-hari Vincent Salon untuk dapat melakukan pemesanan dan penjadwalan jasa di salon secara online.
